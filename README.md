# Budget-Tracker

## Purpose

To provide the avid traveler an app to be able to track their withdrawals and deposits with or without a data/internet connection so that their account balance is accurate when they are traveling.

---

## Technology Used

• Compression
• Mongoose
• Express
• Manifest
• Morgan
• Javascript
• Bootstrap
• Node
• HTML
• CSS
